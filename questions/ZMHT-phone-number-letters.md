# Convert a phone number to all possible numpad strings

### Difficulty

Medium

### Input:

The input is a string containing n numbers eg: "23".

### Output

Output should be an array of all possible letters composing that number.

```
["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].
```
